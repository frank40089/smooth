# smooth
アニメーションのスムージングを行うAfterEffects用のプラグインです。

もともと有償販売していましたがApache2.0ライセンスでオープンソース化しました！
機能は有償販売していた時のままですが、AfterEffects CC2017で利用できるように64bit化なども行っています。
